## Getting Started

Welcome to the Pokémon League Database!
In order to use this program efficiently, enter a proper menu option for each
program input and follow any instructional errors that may arise.

Simple Instructions:
1. Run PokémonLeague.java
2. Enter menu switch case
3. Use inputs to control program
4. Press 6 to end program

General Notes:

Numbers cannot be given String Inputs
String Inputs cannot be empty
Decimals are rounded to the hundredths place
Two trainers need to be in the LeagueDatabase in order for a battle to take place
A maximum of 100 trainers can be created at any given time
Commented Code for Pre-Written Input trainers is available in the implementation
